This repository contains my code and report
for the assignment questions supplied for the
extra credit requirements for the
Introduction to Large Language Models course.

The report is typeset using LaTeX, with 
source code available in [code/](https://github.com/nakulbhat/5HON_LLMAssignment/tree/main/code/) directory.
Implementation was carried out with the help
of a REPL environment, and the code is available
as plain-text Python files in [report](https://github.com/nakulbhat/5HON_LLMAssignment/tree/main/report/) directory.

Due to the limitations of plain-text code files,
only simple comments have been directly included.
Additional, extensive commentaries are provided
as appendices in the report as and when required.
