\begin{abstract}
    This report covers the topics presented in
    the ``Introductions to Large Language Models''
    course taught by Tanmoy Chakraborty and
    Soumen Chakrabarti, available on NPTEL.
    As a graded assignment, the report follows a
    set of seven questions, and accompanies a code
    base available in the repository linked below.
    The report is split into seven sections, each
    corresponding to a question from the assignment.
    Questions will be quoted in full and italicized at the
    start of each section, followed 
    by explanations and code listings where appropriate.
\end{abstract}
